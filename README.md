# PricePulse | Social Price Comparison 💎🛰️

Welcome to the **PricePulse** ecosystem—a gamified, social, and community-driven price scouting platform designed for the modern "Pulse Hunter."

## 🚀 The Pulse Pipeline (Integrated)
This workspace is integrated with GitHub. Every time a new find is pushed to this repository:
1.  **Pulse Verification**: GitHub Actions automatically checks the code for stability.
2.  **Live Deployment**: The platform is instantly updated on **GitHub Pages**.
3.  **Global Radar**: Your community deals become available to the world.

## 🛠️ Tech Stack
*   **Engine**: Vanilla ES6+ JavaScript (Master-Pulse Elite Edition)
*   **Mapping**: Leaflet.js + OpenStreetMap (CartoDB Dark)
*   **Aesthetics**: Charcoal & Electric Lime ("Fun-Pulse" Design Language)
*   **Identity**: LocalStorage Persistent Auth & Identity Hub
